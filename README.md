# Description:
  The project "Frontend-project-lvl1" contains five easy games, launch commands you can find lower.
  Have some fun).

## Setup

```sh
$ make install
```

## Commands to launch games:

```sh
$ brain-calc
```
```sh
$ brain-gcd
```
```sh
$ brain-even
```
```sh
$ brain-prime
```
```sh
$ brain-progression
```
